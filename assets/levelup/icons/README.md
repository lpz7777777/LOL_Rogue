# 升级卡片图标

将以下 PNG 图标放入此目录即可在升级选择界面中显示。

## 所需文件（64x64 或 128x128 推荐）

| 文件名 | 对应强化 | 推荐图标 |
|--------|----------|----------|
| attack.png | AD 加成（攻击之力、全能之力） | 双剑/斧头 |
| health.png | 生命上限 | 心形 |
| speed.png | 攻速、移速 | 疾跑/靴子/速度 |
| range.png | 攻击距离 | 准星/瞄准 |
| magic.png | Q/W/E/R 技能伤害 | 法球/魔法 |

## 免费图标来源（CC0 可商用）

- **OpenGameArt - Simple Stat Icons**  
  https://opengameart.org/content/simple-stat-icons  
  含：双剑(attack)、心(health)、药水(magic)、脚印(speed)、盾牌、准星(range)  
  下载 7z 解压后，将对应 PNG 重命名为上述文件名即可。

- **Kenney Game Icons**  
  https://kenney.nl/assets/game-icons  
  多种游戏图标，可按需挑选并重命名。

将 PNG 文件放入本目录后，无需重启项目，Godot 会自动导入。
